# tech-challenge-2

## Como rodar?
- Instale os pacotes das aplicações `shell`, `alth` e `transaction` rodando o comando:

  ```shell
  pnpm install
  ```

- Suba as aplicações  `shell`, `alth` e `transaction` rodando o comando:

  ```shell
  pnpm dev
  ```

- Abra o endereço `http://localhost:3000` no navegador
